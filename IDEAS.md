# 🚀 Auro Celestial OS - Ý tưởng phát triển tương lai

Danh sách các tính năng tiềm năng được đề xuất bởi người dùng.

## 🎵 1. Hệ thống Âm nhạc (Music Player)
*   **Mô tả:** Vừa chat vừa nghe nhạc. Nhập link YouTube để phát nhạc nền.
*   **Độ khó:** Trung bình (Dễ làm trước).
*   **Ghi chú:** Có thể dùng YouTube IFrame API hoặc một widget nhỏ nổi trên màn hình chat.

## 💼 2. Hệ thống Sự nghiệp (Career System)
*   **Mô tả:** User có thể kinh doanh hoặc đi làm thuê. Giao diện dạng Pixel/Top-down tương tự như tính năng Tổ Ấm.
*   **Độ khó:** Khá (Cần thiết kế logic kinh tế và giao diện riêng).
*   **Ghi chú:** Có thể bắt đầu bằng việc cho User "đi làm" để kiếm tiền (Currency) trong game.

## ⚔️ 3. Hệ thống Chiến đấu (Combat System)
*   **Mô tả:** Game chiến đấu, đánh lộn giữa User và NPC hoặc quái vật.
*   **Độ khó:** Cao.
*   **Ghi chú:** Cần hệ thống chỉ số (HP, ATK, DEF), kỹ năng và hiệu ứng hình ảnh.

## 📹 4. Video Call cho Character
*   **Mô tả:** Gọi video trực tiếp cho nhân vật AI.
*   **Độ khó:** Trung bình - Khá.
*   **Ghi chú:** Có thể dùng Gemini Multimodal Live API hoặc giả lập video bằng ảnh động/Live2D.

## 💰 5. Nạp tiền & Xem quảng cáo
*   **Mô tả:** Hệ thống kinh tế thực tế để duy trì server hoặc mở khóa tính năng cao cấp.
*   **Độ khó:** Trung bình.
*   **Ghi chú:** Tích hợp cổng thanh toán hoặc hệ thống Reward Ads.

## 🖋️ 6. Cài đặt Cá nhân hóa (Typography)
*   **Mô tả:** Đổi kiểu chữ (Font family), cỡ chữ (Font size) cho toàn bộ ứng dụng.
*   **Độ khó:** Dễ (Nên làm sớm).
*   **Ghi chú:** Lưu vào `AppSettings` và áp dụng CSS variables toàn cục.

## 📜 7. Chế độ Ký kết Hệ thống (System Binding)
*   **Mô tả:** Dựa trên các tiểu thuyết hệ thống. Người dùng có thể tìm và ký hợp đồng làm "Ký chủ" cho một Hệ thống AI.
*   **Độ khó:** Khá.
*   **Ghi chú:** Hệ thống sẽ đưa ra nhiệm vụ, phần thưởng và có tính cách riêng (thường là lạnh lùng hoặc tinh quái).

---

### 🛠️ Đề xuất lộ trình (Priority):
1.  **Cài đặt Typography** (Dễ nhất, cải thiện trải nghiệm đọc).
2.  **Hệ thống Âm nhạc** (Tăng cảm xúc khi chat).
3.  **Sự nghiệp (Cấp độ 1)** (Cho user cách kiếm tiền để mua đồ Tổ Ấm).
4.  **Video Call** (Tăng độ chân thực).
